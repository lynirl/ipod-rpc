# ipod-rpc
personal Discord Rich Presence for iTunes/iPods
Windows only, requires an application at the [Discord Developer Portal](https://discord.com/developers/home)
[![pypresence](https://img.shields.io/badge/using-pypresence-00bb88.svg?style=for-the-badge&logo=discord&logoWidth=20)](https://github.com/qwertyquerty/pypresence)

![example of the rich presence on a Discord profile](https://file.garden/ZeMEWj685jeI5jh2/git/1.PNG)

![second example of the rich presence on a Discord profile](https://file.garden/ZeMEWj685jeI5jh2/git/2.PNG)

- shows album covers (uploaded to catbox, cached afterwards), shows placeholder image if not found
- shows which iPod is currently streaming, shows iTunes if the song is from there and not the iPod itself
- has a few issues with keeping track of where you are on a song if you move to another part, but overall knows where you are, when you pause, etc
